Pasos para ejecutar el script:

1. Entrar a la "Evaluación Docente de la UASD".
2. Dar click al botón de "Completar" ubicado en la pantalla.
3. Abrir la consola del Navegador (El código fue creado en 30 minutos usando Claude Code, no hay intención de malware ni nada por el estilo).
4. Copiar el código llamado "Script.js".
5. Pegar el código en la consola.
6. Presionar "Enter" en el teclado.

¿Qué hace el código?
Completa las primeras 30 preguntas de "Evaluación Docente de la UASD", suponiendo que las primeras preguntas sólo tienen "Siempre o Muy Bueno".
